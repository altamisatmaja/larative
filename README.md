## Cara pakai?

1. Navigate to your directory `XAMPP/htdocs/`
2. Clone this repository by typing `https://github.com/altamisatmaja/oh-my-native.git`
3. Run `npm i` in your terminal
4. Execute `npm run artisan` to watch `TailwindCSS`
